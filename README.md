# Projeto Site de Notícias

Este projeto será desenvolvido em NodeJS com Express.JS, fazendo integração com banco de dados **MongoDB**.

## Observações

Apesar de trabalharmos muito com a parte de **Desenvolvimento Web** neste projeto, não voltei meu foco para este, uma vez que existe um outro curso voltado apenas para isto, onde irei entender toda a estrutura de um site, apesar de já saber o básico, e ter desenvolvido outros projetos, o foco do curso de **Desenvolvimento Web** terá um toque muito mais profissional, uma vez que o intuito é me tornar **FullStack**. 

Então, mesmo com o **Layout** bonito, meu foco estará voltado para o backend, e quando iniciar o curso de **Desenvolvimento Web**, não apenas entenderei toda a estrutura como um todo, mas também estarei apto a construir sites de maneira muito mais robusta e cada vez mais profissional.
